# OpenAI Realtime API Python Implementation Demo

基于Python的简单OpenAI实时API应用示例

零延迟对话，可以随时打断。

后续会添加更多功能，如function call等。

## 使用前准备

在项目目录下创建`.env`文件，写入你的OpenAI apikey：

```
OPENAI_API_KEY=sk-xxx
```

## 安装依赖

```shell
pip install -r requirements.txt
```

## 运行

```shell
python main.py
```
